Подробное пояснение по блокам диаграммы:
1. Регистрация и авторизация
Пользователь может либо зарегистрироваться, подтвердив email, либо сразу авторизоваться.
Блок управляет безопасным доступом к системе.

2. Управление моделями
Пользователь создаёт экономическую модель, задавая её параметры и описание.
Возможно загрузить документацию для AI-анализа.
AI анализирует документацию и извлекает фактические данные, сравнивая их с идеализированной моделью.
При необходимости модель корректируется вручную.

3. Расчёт моделей
Пользователь запускает расчёты, выбирая модель и входные данные.
Результаты вычислений доступны пользователю для анализа.
Возможна настройка визуализации для удобного восприятия данных.
Пользователь может экспортировать отчёты и диаграммы.

4. Имитационное моделирование
Пользователь выбирает тип симуляции и загружает финансовую отчётность.
AI на основе загруженных данных формирует модель поведения бизнеса.
Пользователь создаёт и настраивает агентов.
Запускает симуляцию, анализирует результаты и использует AI для генерации рекомендаций по улучшениям.
Может выполнить повторную настройку и симуляцию, если необходимо улучшить результаты.
Доступен экспорт итоговых данных.
